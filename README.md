# QA Jobs Scraper

HTTP-only Node.js scraper for QA/testeur freelance jobs.

- Backend: Express + Cheerio + Axios
- Scheduler: node-cron
- Storage: JSON (Supabase integration READY, see docs)
- Deploy: Render free tier

## Sources (HTTP)
- Free-Work
- Freelance-Informatique
- Codeur.com
- HelloWork
- Jooble
- InFreelancing
- Indeed

## Run
npm start
npm run scrape
npm run cron
